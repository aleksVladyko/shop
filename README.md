# shop
## установите переменные окружения 
   в папке client/.env 
  ### VITE_API_URL=
  ### VITE_LIMIT=
  в папке server/.env 
  ### PORT=

  ## запуск проекта 
  |  client  |  server  |
  | ----------- | ----------- |
| cd client     | cd server  |
| npm init    | npm init  |
| npm run dev   | npm run dev   |
